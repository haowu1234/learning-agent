## 📄 **基于层次图结构的 Agent 记忆框架（GAM）**
（GAM: Hierarchical Graph-based Agentic Memory for LLM Agents）

🔗 https://arxiv.org/abs/2604.12285

💡 **核心贡献**
GAM 提出将记忆编码与记忆整合显式解耦，通过"事件进展图 + 主题关联网络"两层结构同时兼顾快速感知新信息与稳定保留旧知识。相比统一流式记忆或离散结构记忆，GAM 有效避免了短暂噪声对长期知识的干扰。

🔧 **技术亮点**
- 事件进展图（Event Progression Graph）独立存储当前对话轮次，语义发生漂移时才合并进主题关联网络
- 提出图引导的多因子检索策略（Graph-guided Multi-factor Retrieval），提升上下文检索精度
- 层次解耦设计显著降低跨轮次记忆干扰，保持长程一致性

📊 **实验结果**
在 LoCoMo 和 LongDialQA 两个长对话基准上，GAM 在推理准确率与效率两项指标上均优于当前最优基线，论文包含 18 页、6 幅图表。

🎯 **与 Agent/Routing 的关联**
直接解决 Agent 长期交互中的"遗忘与干扰"难题，是构建可靠长链任务 Agent 的核心基础设施。

🔧 **技术亮点**

- 梳理软件工程、企业工作流、GUI、移动系统四大应用场景中的代表性工作
- 指出从单工具调用到多工具长链路编排（long-horizon orchestration）是当前核心研究转变

📊 **实验结果**

## 📄 **ODAR：基于主动推断的 LLM 自适应路由框架**
（ODAR: Principled Adaptive Routing for LLM Reasoning via Active Inference）

🔗 https://arxiv.org/abs/2602.23681

💡 **核心贡献**
ODAR 将 LLM 推理路由问题建模为主动推断（Active Inference）框架，通过难度估计器动态将查询路由到"快速启发式 Agent"或"慢速审慎 Agent"，并用自由能原则（Free Energy Principle）替代 ad-hoc 投票进行答案融合。

🔧 **技术亮点**
- 难度估计器基于摊销主动推断（Amortized Active Inference），实现查询难度的在线评估
- 自由能融合机制（Free Energy Fusion）：最小化变分自由能，平衡对数似然与认知不确定性（varentropy）
- 在 Llama 4 + DeepSeek 全开源栈上验证可复现性

📊 **实验结果**
在 23 个基准上验证，MATH 准确率 98.2%，Humanity's Last Exam (HLE) 54.8%；与同等计算量的均匀采样相比，计算成本降低 82%，准确率仍保持领先。

🎯 **与 Agent/Routing 的关联**
核心即 LLM Routing 论文，从认知科学视角为自适应模型路由提供了理论基础，是该领域最具原理性的工作之一。

## 📄 **AdaptEvolve：进化式 AI Agent 中的自适应 LLM 选择**
（AdaptEvolve: Improving Efficiency of Evolutionary AI Agents through Adaptive Model Selection）

🔗 https://arxiv.org/abs/2602.11931

💡 **核心贡献**
在进化式多轮推理 Agent 框架中，AdaptEvolve 利用模型自身生成置信度（Intrinsic Generation Confidence）作为实时可解度估计，动态选择"足够强大"而非"最强大"的 LLM，在效率与准确率之间找到更优 Pareto 前沿。

🔧 **技术亮点**
- 无需外部控制器：利用模型置信度信号作为路由决策依据，去除额外标注成本
- 模型级联（Model Cascade）机制：按置信度阈值升级到更强模型，避免全局使用大模型
- 适用于"每步都需调用 LLM"的进化式 Agent，解决重复推理的成本爆炸问题

📊 **实验结果**
在多个基准上，置信度驱动的路由策略平均减少推理总成本 37.9%，同时保留静态大模型基线 97.5% 的准确率上界。

🎯 **与 Agent/Routing 的关联**
将 LLM Routing 嵌入 Agent 推理循环（每步动态选模型），是 Agent × Routing 交叉的直接落地方案。

## 🧠 今日洞察

**趋势一：记忆架构走向"图结构 + 层次解耦"。** 单一向量存储或扁平 KV 已难以满足 Agent 长期交互需求，GAM 等工作预示着认知科学启发的分层图记忆将成为主流范式。

**趋势二：LLM Routing 从启发式走向原理性。** ODAR 引入自由能原则、AdaptEvolve 利用生成置信度，均在向"有理论支撑的自适应路由"迈进，而非简单的规则级联——这意味着路由本身正演变为一个一等公民的研究问题。

**趋势三：Agent 规模化的瓶颈正从"能力"转移到"效率与可靠性"。** 工具编排综述和多篇路由论文的密集涌现，说明社区已从"LLM 能不能 Use Tool"跨越到"如何高效、安全、可验证地协调多工具长链路任务"，这是 2026 年 Agent 研究的核心战场。
