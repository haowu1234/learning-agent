### 1. 自主 LLM 智能体的记忆机制：机制、评估与前沿进展
**Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers**

🔗 https://arxiv.org/abs/2603.07670

💡 **核心贡献**
本文是首篇系统覆盖 2022–2026 早期 LLM Agent 记忆领域的综述，将 Agent 记忆形式化为"写-管-读"（write–manage–read）闭环，并从时间跨度、表示底层和控制策略三个维度建立分类体系。研究揭示了当前记忆系统在多会话 Agent 测试中的持久性缺口，并提出连续整合、因果检索、可信反思等开放挑战。

🔧 **技术亮点**
- 五大记忆机制家族：上下文压缩、检索增强存储（RAG）、反射式自我改进、层次化虚拟上下文、策略学习式管理
- 评估维度从静态回忆 benchmark 升级到多会话 agentic 测试（将记忆与决策交织）
- 工程层面覆盖写路径过滤、矛盾处理、延迟预算与隐私治理

📊 **实验结果**
分析了 4 个近期 benchmark，揭示现有系统在长程一致性和持续学习方面存在系统性缺口。

🎯 **与 Agent/Routing 的关联**
为 Agent 系统的记忆模块设计提供了迄今最完整的分类框架，是构建长期任务 Agent 的必读综述。

### 2. 面向长程 LLM 智能体的层次化长期记忆（HiMem）
**HiMem: Hierarchical Long-Term Memory for LLM Long-Horizon Agents**

🔗 https://arxiv.org/abs/2601.06377

💡 **核心贡献**
提出 HiMem 框架，通过"情节记忆（Episode Memory）+ 笔记记忆（Note Memory）"两层层次结构，解决持续交互场景下记忆适应性差、可扩展性弱、无法自我演化的核心痛点。框架中两种记忆类型语义相连，形成从具体事件到抽象知识的层次桥梁。

🔧 **技术亮点**
- Topic-Aware 事件-惊喜双通道分割（Event-Surprise Dual-Channel Segmentation）构建认知一致的情节记忆
- 多阶段信息抽取 pipeline 构建稳定知识的 Note Memory
- 冲突感知的记忆再巩固（Memory Reconsolidation）机制，基于检索反馈动态修订记忆

📊 **实验结果**
在长程对话 benchmark 上，HiMem 在准确性、一致性和长程推理三个维度持续优于代表性基线，同时保持良好的效率表现。

🎯 **与 Agent/Routing 的关联**
直接解决长程 Agent 的核心内存瓶颈，为多轮任务 Agent 的记忆持久化提供了可落地的工程方案。代码开源：https://github.com/jojopdq/HiMem

### 3. 多智能体协调基准测试：Agentic Web 场景（AgentWebBench）
**AgentWebBench: Benchmarking Multi-Agent Coordination in Agentic Web**

🔗 https://arxiv.org/abs/2604.10938

💡 **核心贡献**
引入 AgentWebBench，首个专门评估"用户 Agent ↔ 内容 Agent"去中心化协作范式的 benchmark，覆盖网页搜索、推荐、问答、深度研究四类任务。研究发现多 Agent 协调在 QA 任务上可超越集中式检索，但整体仍依赖模型规模与充分的规划引导。

🔧 **技术亮点**
- 评估 7 个先进 LLM × 3 种协调策略的交叉对比，数据点全面
- 发现去中心化访问存在"流量向少数网站集中"的涌现规律（类似现实 Web 的幂律分布）
- 测试时计算扩展（Test-Time Scaling）同时提升交互可靠性和任务性能

📊 **实验结果**
多 Agent 协调在 QA 任务上可追平甚至超越集中式检索；模型规模越大，协调能力与单体检索的差距越小。

🎯 **与 Agent/Routing 的关联**
为多 Agent 系统中的任务分发与协调策略提供了系统性评估框架，揭示了规划能力是 Agent 协调成功的关键瓶颈。代码开源：https://github.com/cxcscmu/AgentWebBench

### 4. 高效 LLM 推理中的动态模型路由与级联：综述
**Dynamic Model Routing and Cascading for Efficient LLM Inference: A Survey**

🔗 https://arxiv.org/abs/2603.04445

💡 **核心贡献**
系统分析跨多个独立训练 LLM 的路由与级联方法（区别于 MoE 内部路由），提出三维概念框架：何时决策（when）、使用何种信息（what）、如何计算（how），揭示实际系统往往是多范式的组合体。研究表明精心设计的路由系统可超越单一最强模型的性能。

🔧 **技术亮点**
- 覆盖 7 大路由范式：查询难度、人类偏好、聚类、不确定性量化、强化学习、多模态、级联
- 三维分析框架（when/what/how）统一了碎片化路由文献的概念体系
- 指出开放挑战：跨异构架构、模态和应用的通用路由机制尚未解决

📊 **实验结果**
综述证明：路由系统在策略合理时可通过组合专项能力超越最强单模型，同时大幅降低推理成本。

🎯 **与 Agent/Routing 的关联**
直接命中 LLM Routing 核心主题，为构建 Agent 系统中的模型选择模块提供了最新综述级参考。
