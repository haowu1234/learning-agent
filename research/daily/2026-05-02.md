📄 **【AgeMem】统一长短期记忆管理的大语言模型智能体**（Learning Unified Long-Term and Short-Term Memory Management for Large Language Model Agents）
🔗 https://arxiv.org/abs/2601.01885
💡 核心贡献：
提出 AgeMem 框架，将长期记忆（LTM）和短期记忆（STM）的管理统一到智能体策略中，通过 tool-based actions 让 LLM 自主决策何时存储、检索、更新、摘要或丢弃信息。采用三阶段渐进强化学习训练策略，设计 step-wise GRPO 解决记忆操作引发的稀疏奖励问题。
🔧 技术亮点：
- 记忆操作作为工具动作暴露给 LLM，实现端到端可优化的记忆管理
- 三阶段渐进 RL 训练 + step-wise GRPO 处理稀疏奖励
- 统一框架消除 LTM/STM 双控制器架构，提升自适应性
📊 实验结果：在5个长视野基准测试上，AgeMem 持续超越强记忆增强基线，任务完成率提升显著，长期记忆质量更高，上下文利用更高效。代码已开源：https://github.com/y1y5/AgeMem（v2于2026-04-30更新）
🎯 与 Agent/Routing 的关联：直接强化 Agent 的长视野推理能力，记忆操作即为一种内部"路由"决策——哪些信息值得保留并路由到未来的推理步骤。

📄 **【记忆综述】自主 LLM 智能体的记忆机制、评估与前沿**（Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers）
🔗 https://arxiv.org/abs/2603.07670
💡 核心贡献：
系统梳理2022-2026年初 LLM 智能体记忆设计、实现与评估的结构化全景综述，将记忆正式化为 write–manage–read 闭环，提出横跨时间范围、表示底层和控制策略的三维分类体系。深度分析五类记忆机制：上下文压缩、检索增强存储、反射式自提升、层次虚拟上下文、策略学习管理。
🔧 技术亮点：
- 三维分类体系：时间范围 × 表示底层 × 控制策略
- 梳理评估从静态召回基准到多会话交织决策的演进
- 覆盖个人助手、编程 Agent、开放世界游戏、科学推理、多智能体协作等应用场景
📊 实验结果：分析4个近期基准（MemGPT-Bench 等），揭示当前系统在持续整合、因果检索、可信反射方面的顽固缺陷。
🎯 与 Agent/Routing 的关联：为 Agent 记忆领域提供权威理论框架，直接指导下一代 Agent 架构设计，尤其是记忆与决策路由的协同优化。

📄 **【Plan-and-Act】长视野任务中智能体规划能力的提升**（Plan-and-Act: Improving Planning of Agents for Long-Horizon Tasks）
🔗 https://arxiv.org/abs/2503.09572
💡 核心贡献：
提出 Plan-and-Act 框架，将高层规划与低层执行分离：Planner 模型生成结构化高层计划，Executor 模型将计划转化为环境动作。引入可扩展合成数据生成方法，通过对 ground-truth 轨迹标注可行计划并用多样化样本增强，有效训练 Planner。
🔧 技术亮点：
- Planner-Executor 解耦架构，各司其职，平衡规划目标与执行细节
- 合成数据生成：从真实轨迹标注计划 + 多样化扩充提升泛化
- Web 导航作为长视野规划代表性环境进行全面评估
📊 实验结果：WebArena-Lite 基准 SOTA 57.58% 成功率；WebVoyager 纯文本模态 SOTA 81.36% 成功率。已被 ICML 2025 接收。
🎯 与 Agent/Routing 的关联：Planner-Executor 分离实质上是一种任务级路由，根据任务复杂度动态决定调用哪层能力，为多 Agent 编排提供可借鉴的分层设计范式。

📄 **【AgentGate】互联网智能体的轻量级结构化路由引擎**（AgentGate: A Lightweight Structured Routing Engine for the Internet of Agents）
🔗 https://arxiv.org/abs/2604.06696
💡 核心贡献：
面向"智能体互联网"生态，提出 AgentGate——将 Agent 路由问题从无约束文本生成重新定义为约束决策问题，分两阶段执行：动作决策（单 Agent 调用/多 Agent 规划/直接响应/安全升级）+ 结构化实例化（生成目标 Agent、结构化参数或多步计划）。配合候选感知的微调方案和难负例训练，使紧凑模型（3B-7B）实现有竞争力的路由性能。
🔧 技术亮点：
- 路由重定义为约束决策，解决开放文本生成的不确定性问题
- 双阶段设计：action decision + structural grounding，逻辑清晰可扩展
- 候选感知监督 + 难负例训练，3B-7B 开源模型即可胜任资源受限部署
📊 实验结果：在专项路由基准测试中，3B-7B 模型展现竞争性能，模型差异主要体现在动作预测、候选选择和结构化接地质量三个维度。
🎯 与 Agent/Routing 的关联：这是本期最直接的 Routing 论文，明确解决多 Agent 生态下的请求分发问题，为隐私感知、低延迟、低成本的 Agent 路由系统提供系统设计参考。

## 🧠 今日洞察

当前 Agent 研究正在经历从"能用"到"可靠"的关键跃迁：**记忆管理**已从附加模块进化为可端到端学习的核心策略（AgeMem），**规划能力**通过 Planner-Executor 解耦和合成数据显著提升（Plan-and-Act）。与此同时，随着"智能体互联网"生态成形，**Agent Routing**（智能体路由）正成为新的系统层挑战——AgentGate 和 UniRoute 等工作将路由从启发式文本匹配提升为可优化的约束决策，轻量级模型已能胜任资源受限场景。综合来看，**记忆 × 规划 × 路由**的三角协同将是下一阶段 Agent 基础设施的核心命题。
