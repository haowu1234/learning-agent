### 1️⃣ 多智能体显式特征推断协调框架（Explicit Trait Inference for Multi-Agent Coordination）

🔗 https://arxiv.org/abs/2604.19278

💡 **核心贡献**
提出 ETI（Explicit Trait Inference）方法，让 Agent 能从交互历史中推断协作伙伴的心理特征（"温暖度"与"能力"两个维度），并据此调整自身决策策略。这是首个系统性验证 LLM Agent 可以可靠推断他者特征、并利用结构化特征感知来改善多 Agent 协调的工作。

🔧 **技术亮点**
- 引入心理学双维度（warmth × competence）作为 Agent 建模他者的认知框架
- 基于交互历史动态维护伙伴特征画像（trait profile），无需额外训练
- 在经济博弈与 MultiAgentBench 两类环境下验证框架通用性

📊 **实验结果**
- 受控经济博弈场景：相比 CoT baseline，payoff loss 降低 **45–77%**
- 复杂多 Agent 场景（MultiAgentBench）：性能提升 **3–29%**（视场景和模型而定）
- ACL 2026 Main Conference 录用

🎯 **与 Agent/Routing 的关联**
ETI 的特征路由思想可直接迁移至 LLM Routing：通过动态感知查询方与模型能力特征来自适应选择最优模型。

### 2️⃣ 长周期企业 AI Agent 四轴决策对齐（Four-Axis Decision Alignment for Long-Horizon Enterprise AI Agents）

🔗 https://arxiv.org/abs/2604.19457

💡 **核心贡献**
针对贷款审核、理赔裁定等高风险长周期 Agent 决策场景，提出将决策行为分解为四个正交对齐维度（FRP/RCS/CRR/CAR），打破单一 task-success 指标掩盖失败模式的问题。新增合规重建轴（CRR）和校准弃权轴（CAR）是学界空白的首次填补。

🔧 **技术亮点**
- 四轴框架：事实精度(FRP)、推理连贯性(RCS)、合规重建(CRR)、校准弃权(CAR)，各自独立可测
- 构建 LongHorizon-Bench 基准，包含贷款资格与保险理赔两类确定性 ground-truth 场景
- 发现"摘要+事实保留提示"方案在 FRP/RCS/CRR 上超越复杂记忆架构

📊 **实验结果**
- 测试 6 种记忆架构；检索型架构在事实精度轴上显著下滑
- 所有架构在决策对齐轴（弃权）上表现一致缺失，揭示现有系统的结构性盲区
- 代码开源：https://github.com/vasundras/decision-alignment-long-horizon-agents

🎯 **与 Agent/Routing 的关联**
四轴框架提供了一种可推广的评估方法论，为 Agent Router 的路由正确性、合规性评估提供了可借鉴的结构化指标体系。

### 3️⃣ 多 Agent 可复用策略学习的生成引擎优化（From Experience to Skill: Multi-Agent Generative Engine Optimization via Reusable Strategy Learning）

🔗 https://arxiv.org/abs/2604.19516

💡 **核心贡献**
将生成引擎优化（GEO）重构为策略学习问题，提出 MAGEO 框架：多 Agent 协作执行规划/编辑/保真度评估，同时将有效编辑模式逐步蒸馏为可复用的、引擎特化的优化技能（Skill），实现跨任务和跨引擎的知识迁移。

🔧 **技术亮点**
- 引入 Twin Branch Evaluation Protocol，实现内容编辑因果归因，区分可见性与引用准确性
- 设计 DSV-CF 双轴指标，统一语义可见度与归因准确率
- 发布 MSME-GEO-Bench：多场景、多引擎、基于真实查询的 GEO 评测基准

📊 **实验结果**
- 在三大主流生成引擎上，MAGEO 显著优于启发式基线
- 消融实验确认：引擎特化偏好建模 + 策略复用是核心增益来源
- ACL 2026 Findings 录用，代码：https://github.com/Wu-beining/MAGEO

🎯 **与 Agent/Routing 的关联**
MAGEO 的"经验→技能"蒸馏机制与 Agent Memory 系统高度同构，为自主 Agent 的长期记忆和策略积累提供了工程参考范式。

### 4️⃣ 多 LLM 动态路由与级联推理综合调研（Dynamic Model Routing and Cascading for Efficient LLM Inference: A Survey）

🔗 https://arxiv.org/abs/2603.04445

💡 **核心贡献**
系统梳理多 LLM 路由与级联方案的最新进展，提出三维概念框架（何时决策 × 利用何种信息 × 如何计算），覆盖查询难度、人类偏好、聚类、不确定性量化、强化学习、多模态、级联等七大路由范式。

🔧 **技术亮点**
- 明确区分"跨独立 LLM 路由"与"模型内部 MoE"，澄清两类架构的边界
- 总结路由系统的组合性特征：实际系统往往是多范式融合而非单一策略
- 指出开放挑战：跨架构、跨模态、跨应用域的路由泛化能力仍待突破

📊 **实验结果**
- 综述论文，无直接实验数据；分析表明：设计良好的路由系统可超越最强单一模型
- 2026年4月21日更新至 v2（最新版本）

本文是目前最全面的 LLM Routing 综述，直接服务于 Router Agent 的技术选型与架构设计决策。

## 🧠 今日洞察

1. **多 Agent 协调从"任务分工"走向"心理建模"**：ETI 的工作表明，Agent 系统的下一个瓶颈不是执行能力，而是相互理解能力——让 Agent 像人一样推断同伴的动机和能力，将大幅降低协调失败率。

2. **长周期决策的评估框架正在补位**：单一成功率指标在复杂 Agent 场景下已经失效，四轴对齐框架（FRP/RCS/CRR/CAR）和路由三维框架代表了"从可用走向可信"的评估方向演进。

3. **路由与记忆的边界正在模糊**：MAGEO 的策略蒸馏、ETI 的特征画像维护、LLM Routing 的动态选择——三者本质上都是"在经验中积累可复用判断"，预示 Memory × Routing 的融合将成为 Agent 基础设施的核心议题。
