### 1. 统一长短期记忆管理的智能体记忆框架

**（AgeMem: Learning Unified Long-Term and Short-Term Memory Management for Large Language Model Agents）**

🔗 https://arxiv.org/abs/2601.01885

💡 **核心贡献**
AgeMem 提出将长期记忆（LTM）和短期记忆（STM）统一集成到 Agent 策略中，通过工具调用形式暴露记忆操作（存储/检索/更新/摘要/遗忘），让 LLM Agent 自主决策记忆行为。采用三阶段渐进式强化学习训练策略，并设计 step-wise GRPO 应对记忆操作导致的稀疏、不连续奖励问题。

🔧 **技术亮点**
- 记忆操作作为 tool-based actions，将 LTM/STM 统一为端到端可优化的 Agent 策略组件
- 三阶段渐进式 RL 训练：解耦记忆行为的学习难度，从基础记忆逐步过渡到复杂策略
- Step-wise GRPO：专门针对记忆操作稀疏奖励设计，修正标准 GRPO 的信用分配偏差

📊 **实验结果**
在 5 个长视野基准（Long-horizon Benchmarks）上，AgeMem 在多个 LLM 骨干上持续超越强记忆增强基线，实现更高任务性能、更高质量的长期记忆和更高效的上下文利用。代码已开源：https://github.com/y1y5/AgeMem

🎯 **与 Agent/Routing 的关联**
直接解决 Long-horizon Agent 的核心瓶颈——记忆管理，为需要跨多步骤维持状态的复杂 Agent 任务提供统一且可训练的记忆系统基础设施。

### 2. 链式推理 × 树状学习：多轮 Agent 策略优化自纠正框架 T-STAR

**（Reason in Chains, Learn in Trees: Self-Rectification and Grafting for Multi-turn Agent Policy Optimization）**

🔗 https://arxiv.org/abs/2604.07165

💡 **核心贡献**
提出 T-STAR（Tree-structured Self-Taught Agent Rectification）框架，通过将 Agent 多步推理轨迹组织为"认知树"结构，恢复独立轨迹间潜在的相关奖励结构，实现步骤级精确信用分配与推理错误自我纠正。

🔧 **技术亮点**
- **认知树（Cognitive Tree）**：识别并合并功能相似的推理步骤，将多条轨迹统一为树结构，揭示隐藏的奖励相关性
- **内省估值机制（Introspective Valuation）**：通过树反向传播轨迹级奖励，获得方差减小的步骤级相对优势，解决稀疏奖励下的信用分配难题
- **上下文思想嫁接（In-Context Thought Grafting）**：在关键分歧点对比成功/失败分支，自动合成纠正性推理；配合 Bradley-Terry 型手术损失进行精准策略优化

📊 **实验结果**
在具身任务、交互任务、推理任务和规划任务基准上均超越强基线，在需要扩展推理链的任务上提升最为显著（v2 于 2026-04-15 发布）。

🎯 **与 Agent/Routing 的关联**
为多轮对话 Agent 的策略优化提供树状学习框架，可直接应用于 Multi-Agent 场景下的协作规划与任务分配策略训练。

### 3. 互联网智能体的轻量级结构化路由引擎 AgentGate

**（AgentGate: A Lightweight Structured Routing Engine for the Internet of Agents）**

🔗 https://arxiv.org/abs/2604.06696

💡 **核心贡献**
将 Agent 路由问题从无约束文本生成重新形式化为约束决策问题，提出双阶段结构化路由架构：动作决策阶段（判断调用单 Agent/多 Agent/直接响应/安全升级）+ 结构化实例化阶段（生成目标 Agent、结构化参数或多步计划），配合候选感知的微调方案实现紧凑模型的竞争性路由性能。

🔧 **技术亮点**
- 候选感知路由：将可用 Agent 候选信息纳入路由决策，从根本上区别于无感知文本生成路由
- 双阶段解耦设计：动作类型预测与结构化实例化分离，各阶段独立优化，支持资源受限部署
- 路由导向微调：含硬负例（Hard Negatives）的路由专项训练数据，3B-7B 开源模型可达竞争性水平

📊 **实验结果**
在专门构建的路由基准上，3B-7B 参数量的开源模型在约束场景中可提供竞争性路由性能；模型差异主要体现在动作预测、候选选择和结构化实例化质量三个维度。

🎯 **与 Agent/Routing 的关联**
直击 Agent Routing 核心问题，将模型路由从 LLM 级扩展到 Agent 级，为多 Agent 系统提供隐私保护、低延迟、低成本的路由引擎方案。

### 4. 通用 LLM 动态路由框架 UniRoute

**（Universal Model Routing for Efficient LLM Inference）**

🔗 https://openreview.net/forum?id=ka82fvJ5f1

💡 **核心贡献**
提出 UniRoute，一种支持动态路由的通用 LLM 推理框架：不同于现有方法固定路由池，UniRoute 允许在测试时添加从未见过的新模型，而无需重新训练路由器，通过将每个 LLM 表示为基于代表性提示预测的特征向量来实现模型无关的泛化。

🔧 **技术亮点**
- LLM 特征向量表示：以代表性提示集上的预测结果为基，将每个 LLM 编码为可比较的特征向量，使路由器对新模型零样本泛化
- 两种路由实例化：基于聚类的路由（Cluster-based Routing）和学习聚类映射（Learned Cluster Map），理论上均为最优路由规则的估计
- 超额风险界（Excess Risk Bound）：提供路由误差的理论量化，为工程实践提供可靠性保证

📊 **实验结果**
在多个公开基准上，UniRoute 成功路由超过 30 个未见 LLM，在准确性和推理成本之间取得显著优于固定池路由的 trade-off。

🎯 **与 Agent/Routing 的关联**
解决 LLM Routing 中的开放世界问题（Open-world Routing），使路由基础设施可随新模型上线动态扩展，对多模型 Agent 系统的自适应推理具有直接应用价值。

## 🧠 今日洞察

**趋势一：记忆与推理深度集成** — Agent 记忆不再是独立模块，而是直接嵌入 RL 策略训练（AgeMem），记忆操作的可训练化是 Long-horizon Agent 能力突破的关键路径。

**趋势二：路由粒度向上延伸** — Routing 研究正从 LLM-level（UniRoute、RTR）向 Agent-level（AgentGate）演进，"互联网智能体"生态的路由调度成为新的系统研究热点，隐私保护与资源约束下的高效分发是核心设计矛盾。

**趋势三：树状结构统一 Agent 优化** — 将线性轨迹重组为树，既解决信用分配（T-STAR），也支持跨轨迹知识复用——树状学习范式预计将成为 Multi-turn Agent 策略优化的主流结构抽象。
