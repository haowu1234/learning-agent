### 1. 面向高效 Agent：记忆、工具学习与规划综述

**原文标题**：Toward Efficient Agents: Memory, Tool learning, and Planning

🔗 https://arxiv.org/abs/2601.14192

💡 **核心贡献**
本文从记忆管理、工具学习和规划三个核心维度系统审视 LLM Agent 的效率问题，填补了当前研究过度关注"有效性"而忽视"效率"的空白。提出了以 Pareto 前沿衡量效率-效果权衡的评估框架，并梳理了 200+ 篇相关工作。

🔧 **技术亮点**
- 通过上下文压缩与管理来约束 context 窗口，降低 Token 消耗
- 引入强化学习奖励信号显式最小化工具调用次数
- 采用受控搜索机制（而非贪心展开）提升规划效率

📊 **实验结果**
综述性工作，整理了效率导向 benchmark 的评估协议与指标体系（延迟、Token 数、步骤数等），无单一实验数字。

🎯 **与 Agent/Routing 的关联**
直接覆盖 Agent 三大核心子系统，是理解高效 Agent 架构的权威参考综述。

### 2. 面向自主 LLM Agent 的记忆机制：机制、评估与前沿展望

**原文标题**：Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers

🔗 https://arxiv.org/abs/2603.07670

💡 **核心贡献**
系统梳理 2022-2026 年初 LLM Agent 记忆领域研究，将记忆形式化为"写入-管理-读取"循环，并提出跨越时间范围、表征基底、控制策略三个维度的分类体系。揭示了静态召回 benchmark 无法反映真实多会话记忆能力的评估瓶颈。

🔧 **技术亮点**
- 五类记忆机制：context 驻留压缩、RAG 存储、反思自我优化、层次虚拟上下文、策略学习式管理
- 评估从静态召回向多会话交错决策测试演进，覆盖4个最新 benchmark
- 明确提出写路径过滤、矛盾处理、延迟预算、隐私治理的工程实践要点

📊 **实验结果**
通过四个近期 benchmark 揭露现有系统在跨会话连续记忆上存在的系统性缺口。

🎯 **与 Agent/Routing 的关联**
记忆系统是长程 Agent 决策的基础设施，与 Agent 的状态管理和任务路由深度绑定。

### 3. 长程企业 AI Agent 的四轴决策对齐

**原文标题**：Four-Axis Decision Alignment for Long-Horizon Enterprise AI Agents

🔗 https://arxiv.org/abs/2604.19457

💡 **核心贡献**
针对贷款审批、理赔裁定等高风险长程决策场景，提出将 Agent 行为分解为四个正交对齐轴（事实精确度 FRP、推理连贯性 RCS、合规重建 CRR、校准弃权 CAR），打破单一任务成功率掩盖深层失效的局限。CRR 是首个以监管为锚点的新型评估维度。

🔧 **技术亮点**
- 构建含确定性 ground-truth 的受控 benchmark：LongHorizon-Bench（贷款资质 + 保险理赔）
- 对比 6 种记忆架构：发现结构化 schema 架构存在"脚手架税"，普通摘要 + 事实保留 prompt 反而是强 baseline
- 引入 CAR 轴，首次将"是否应该弃权决策"与准确率分离测量

📊 **实验结果**
6 种记忆架构测试：检索架构在事实精确度（FRP）上崩溃；所有 6 架构在每个 case 均强制给出决策，暴露了 CAR 轴的普遍性失效；摘要方案在 FRP/RCS/CRR 上逆转了预先假设。

🎯 **与 Agent/Routing 的关联**
为长程 Agent 任务路由中的"何时拒绝/转交"问题提供了量化框架，是企业级 Agent 决策路由的方法论基础。

### 4. 显式特征推断用于多智能体协调（ACL 2026 收录）

**原文标题**：Explicit Trait Inference for Multi-Agent Coordination

🔗 https://arxiv.org/abs/2604.19278

💡 **核心贡献**
提出 ETI（显式特征推断）方法，让 Agent 从交互历史中显式推断协作伙伴的"温暖度"（信任）与"能力度"（技能）两个心理维度，以此指导协调决策，从根本上解决多 Agent 系统中的目标漂移与行为失配问题。

🔧 **技术亮点**
- 将社会心理学"温暖-能力"双维度模型嵌入 Agent 决策循环
- 在经济博弈受控场景中验证：协调收益损失减少 **45-77%**
- 在 MultiAgentBench 复杂场景中相较 CoT baseline 提升 **3-29%**（依场景和模型不同）

📊 **实验结果**
经济博弈（收益损失 ↓45-77%）+ MultiAgentBench（性能 ↑3-29%）；特征推断 profile 与 Agent 实际行为高度相关，验证了机制有效性。已被 **ACL 2026 Main Conference** 接收。

🎯 **与 Agent/Routing 的关联**
为多 Agent 系统中的动态任务分配与智能路由提供了基于"伙伴能力感知"的新范式，是 Agent Routing 的社会性扩展。

## 🧠 今日洞察

1. **记忆从可选变为基础设施**：三篇论文（综述 + 记忆机制调研 + 四轴对齐）都将记忆系统置于核心位置，标志着 Agent 架构中"有状态记忆"已从实验特性演进为工程必需品，设计时应优先考虑记忆层而非事后补打。

2. **评估体系加速细化**：单一任务成功率正被多维度指标取代（四轴对齐的 FRP/RCS/CRR/CAR，ETI 的双维度心理特征），这预示着 2026 年 Agent benchmark 将迎来一次"评估范式升级"，粗粒度 leaderboard 将难以反映系统真实能力。

3. **Multi-Agent 协调走向"社会感知"**：ETI 将心理学特征推断引入 Agent 协调，LLM Routing 的下一个前沿不仅是"选哪个模型"，更是"如何感知协作伙伴能力并自适应调度"——这将推动 Routing 与 Multi-Agent 研究的深度融合。
