### 1. 超越RAG的Agent记忆：解耦与聚合检索

**原文标题**：Beyond RAG for Agent Memory: Retrieval by Decoupling and Aggregation (xMemory)

🔗 https://arxiv.org/abs/2602.02007

💡 **核心贡献**
标准RAG对大规模异构语料有效，但不适合Agent记忆场景——对话流高度相关、存在大量重复片段。本文提出xMemory，将记忆解耦为语义组件并组织成层次结构，以稀疏-语义目标引导节点分裂与合并，实现自顶向下的精准多事实检索。

🔧 **技术亮点**
- 记忆分层结构：主题（themes）→ 语义（semantics）→ 片段（episodes）→ 原始消息，逐层精细化
- 稀疏-语义双目标函数：在紧凑性（避免冗余）和语义保真之间求均衡
- 自适应展开策略：只有在能降低读者不确定性时才向下展开到原始消息

📊 **实验结果**
在 LoCoMo 和 PerLTQA 基准上，测试最新3款LLM，答案质量和Token效率均有一致提升（相比标准RAG减少冗余上下文）。

🎯 **与Agent/Routing的关联**
直接解决长对话Agent的记忆检索瓶颈，是构建持久记忆型Agent的核心基础设施组件。

### 2. 大型工具空间中基于熵引导分支的长程任务执行

**原文标题**：Long-Horizon Plan Execution in Large Tool Spaces through Entropy-Guided Branching (EGB + SLATE)

🔗 https://arxiv.org/abs/2604.12126

💡 **核心贡献**
在超大型工具库（电商场景）中进行多步任务规划存在两大瓶颈：缺乏严格的计划级评估框架，以及大决策空间带来的计算开销。本文同时贡献了新基准SLATE和不确定性感知搜索算法EGB，双管齐下。

🔧 **技术亮点**
- SLATE基准：大规模上下文感知的电商API工具集评估框架，支持多样化但功能等价的执行轨迹
- 熵引导分支（EGB）：在预测熵高的节点动态扩展决策分支，优化探索-利用权衡
- 揭示现有Agent在自我纠错和搜索效率上的系统性弱点

📊 **实验结果**
SLATE上的实验表明EGB显著提升任务成功率和计算效率；发现当前主流Agent在大工具空间中均有明显的搜索效率缺陷。

🎯 **与Agent/Routing的关联**
EGB本质是一种在工具调用决策树中的动态路由机制，将不确定性量化引入Agent的行动选择。

### 3. 高效长程任务规划助力多Agent协作

**原文标题**：ELHPlan: Efficient Long-Horizon Task Planning for Multi-Agent Collaboration

🔗 https://arxiv.org/abs/2509.24230

💡 **核心贡献**
多机器人协作中，开环方法缺乏适应性，迭代方法计算代价随团队规模指数增长。ELHPlan引入"意图绑定行动链"作为基本规划原语，在适应性与效率之间找到新的平衡点。

🔧 **技术亮点**
- 行动链（Action Chains）：将行动序列显式绑定到子目标意图，提供更长的前瞻视野而无需完全重规划
- 循环式规划流程：构建 → 主动验证冲突/可行性 → 定向修复 → 执行
- 提倡综合效率指标：同时考量Token消耗和规划时间，而非仅看任务成功率

📊 **实验结果**
在TDW-MAT和C-WAH基准上，ELHPlan以可比的任务成功率，仅消耗SOTA方法 **30-40%** 的Token。

🎯 **与Agent/Routing的关联**
展示了如何在多Agent系统中通过结构化规划原语实现计算资源的智能分配，与Routing中的"按需计算"思想高度吻合。

### 4. 基于自由能原理的LLM推理自适应路由

**原文标题**：Principled Adaptive Routing for LLM Reasoning via Active Inference (ODAR-Expert)

🔗 https://arxiv.org/abs/2602.23681

💡 **核心贡献**
当前LLM推理范式正从参数规模转向测试时计算规模，但统一采样（best-of-N、自洽）代价高且易引发"过度思考"。ODAR-Expert提出基于主动推断的查询难度估计器，将查询动态路由给快速启发式Agent或深思熟虑型Slow Agent。

🔧 **技术亮点**
- 摊销主动推断（Amortized Active Inference）：实时估计查询难度，O(1)推断开销
- 变分自由能融合：以最小化变分自由能为目标选择答案，平衡对数似然与认知不确定性（varentropy）
- 完全开源验证：Llama 4 + DeepSeek 栈上复现，计算成本降低 **82%**

📊 **实验结果**
跨23个基准测试，MATH准确率达 **98.2%**，Humanity's Last Exam(HLE)达 **54.8%**；相比同构采样，在计算匹配条件下持续改善精度-效率前沿。

🎯 **与Agent/Routing的关联**
这是LLM Routing的标杆性工作：用主动推断原理决定"用哪个Agent/多少算力"，直接回答Routing的核心问题。
