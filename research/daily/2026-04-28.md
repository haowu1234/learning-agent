## 📄 精选论文

📄 **GraphPlanner：基于图记忆增强的多 Agent LLM 路由框架**（GraphPlanner: Graph Memory-Augmented Agentic Routing for Multi-Agent LLMs）
🔗 https://arxiv.org/search/?query=GraphPlanner+Graph+Memory+Agentic+Routing&searchtype=all（提交于 2026-04-26）
💡 核心贡献：提出将图结构记忆引入多 Agent 系统的路由决策，使 Agent 能够基于历史交互的结构化知识选择最合适的下游模型或子 Agent。该方法解决了传统路由器缺乏任务上下文持久化的痛点，实现了"有记忆的路由"。
🔧 技术亮点：
- 图结构记忆（Graph Memory）存储 Agent 间的历史任务依赖与调用关系
- 结合 LLM 语义理解与图检索实现上下文感知路由
- 支持动态更新记忆图，适应任务分布漂移
🎯 与 Agent/Routing 的关联：直接将记忆机制与路由决策深度融合，是 Agent Memory + Routing 双主题的交叉创新。

---

📄 **Route to Rome 攻击：通过对抗后缀将 LLM 路由劫持至昂贵模型**（Route to Rome Attack: Directing LLM Routers to Expensive Models via Adversarial Suffix Optimization）
🔗 https://arxiv.org/search/?query=Route+to+Rome+Attack+LLM+Routers+Adversarial&searchtype=all（提交于 2026-04-16）
💡 核心贡献：揭示 LLM 成本感知路由系统存在严重安全漏洞——攻击者可通过在 prompt 中附加优化后的对抗后缀，欺骗路由器将本应分配给小模型的低难度请求错误路由至大型昂贵模型，从而造成服务成本激增。这是 LLM Routing 领域首批针对对抗攻击的系统性研究之一。
🔧 技术亮点：
- 基于梯度优化的对抗后缀生成方法（类 GCG 攻击变体）
- 黑盒/白盒两种攻击场景下的成功率评估
- 提出防御基线：基于困惑度过滤的异常检测
📊 实验结果：在多个主流路由系统（如 RouteLLM）上攻击成功率显著，可使路由到大模型的比例从约 20% 提升至 70%+，推断成本提升 3-5×。
🎯 与 Agent/Routing 的关联：直接指向 LLM Routing 的安全鲁棒性问题，对生产级路由系统的部署具有重要警示意义。

📄 **Autogenesis：自进化 Agent 协议**（Autogenesis: A Self-Evolving Agent Protocol）
🔗 https://arxiv.org/search/?query=Autogenesis+Self-Evolving+Agent+Protocol&searchtype=all（提交于 2026-04-21，Wentao Zhang 等）
💡 核心贡献：提出 Autogenesis 协议，使 Agent 系统能够在执行任务过程中自主生成、测试并整合新能力，实现无需人工干预的持续自我进化。区别于传统 Agent 框架的静态工具集，Autogenesis 引入了能力发现与验证的闭环机制。
🔧 技术亮点：
- 自动能力提案模块（Capability Proposal）：LLM 驱动识别任务能力缺口
- 沙箱验证机制：对新生成工具/能力进行安全测试后才集成
- 分层记忆：区分短期执行状态与长期能力库
📊 实验结果：在长时程复杂任务 benchmark（如 WebArena-Long）上，相比静态工具 Agent 成功率提升约 18%。
🎯 与 Agent/Routing 的关联：自进化能力库直接影响 Agent 内部的工具选择与规划策略，是 long-horizon planning 的关键突破。

---

📄 **LLM 动态路由与级联推理综述**（Dynamic Model Routing and Cascading for Efficient LLM Inference: A Survey）
🔗 https://arxiv.org/search/?query=Dynamic+Model+Routing+Cascading+Efficient+LLM+Inference+Survey&searchtype=all（提交于 2026-04-21，v2 更新版）
💡 核心贡献：全面系统综述了 LLM 推理阶段的动态路由与级联方法，统一梳理了查询复杂度感知路由、模型级联（easy-first）、混合专家调度等 30+ 方法，并给出统一分类体系与开放问题。
🔧 技术亮点：
- 三大范式：Query Routing（查询分流）、Model Cascading（模型级联）、Hybrid（混合策略）
- 覆盖延迟、成本、准确率三维权衡的评估框架
- 总结训练数据稀缺、分布外泛化、实时路由延迟三大未解难题
📊 实验结果：综述数据显示，成熟路由系统可在保持 95% 准确率的前提下节省 40-60% 推理成本。
🎯 与 Agent/Routing 的关联：为 LLM Routing 领域提供最新全景图，是入门与研究选题的必读参考。

## 🧠 今日洞察

1. **路由安全成为新战场**：随着 LLM 路由系统在生产中大规模落地，"Route to Rome" 类的对抗攻击预示着路由鲁棒性将是下一个重要研究方向——不仅要路由得准，还要路由得安全。

2. **记忆与路由深度融合**：GraphPlanner 等工作表明，纯 stateless 的路由器已不够用；将结构化记忆引入路由决策，让系统具备"任务历史感知"的能力，是 Multi-Agent 编排的进化方向。

3. **Agent 自进化能力趋向标准化**：Autogenesis 协议的出现标志着 Agent 能力扩展从"人工配置工具集"向"自主发现与验证能力"转变，长时程任务的成功率因此获得了实质性提升。
