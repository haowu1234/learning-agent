## 📄 Paper 1

📄 **[AgeMem：统一长短期记忆管理的 LLM Agent 框架]**（Learning Unified Long-Term and Short-Term Memory Management for Large Language Model Agents）
🔗 https://arxiv.org/abs/2601.01885

💡 **核心贡献**
提出 AgeMem 框架，将长期记忆（LTM）与短期记忆（STM）统一为 Agent 策略中可直接操作的工具型动作，摆脱了传统依赖启发式规则或辅助控制器的局限；通过三阶段渐进式强化学习 + 逐步 GRPO 训练，解决记忆操作带来的稀疏奖励问题。

🔧 **技术亮点**
- 将 store/retrieve/update/summarize/discard 等记忆操作暴露为工具调用，由 LLM 自主决策执行时机
- 三阶段渐进式 RL 训练策略，逐步解锁不同难度记忆管理行为
- 提出 Step-wise GRPO，专门应对记忆操作导致的非连续稀疏奖励信号

📊 **实验结果**
在 5 个长视野基准测试上，AgeMem 持续优于强基线记忆增强方法，任务完成率、记忆质量和上下文利用效率均有提升，且在多种 LLM 主干上具有泛化性。

🎯 **与 Agent/Routing 的关联**
直接推进了 Agent 长视野记忆管理的端到端可优化性，是构建持久化 Agent 的关键基础组件。

## 📄 Paper 2

📄 **[自主 LLM Agent 记忆综述：机制、评估与前沿挑战]**（Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers）
🔗 https://arxiv.org/abs/2603.07670

💡 **核心贡献**
系统梳理 2022–2026 年初 LLM Agent 记忆领域全貌，建立「写入-管理-读取」循环的形式化框架，并提出覆盖时间维度、表示基底、控制策略三个维度的分类体系；同时分析了记忆评估从静态 Recall 到多会话 Agentic 测试的范式迁移。

🔧 **技术亮点**
- 五大记忆机制族：上下文压缩、检索增强存储、反射式自改进、层次化虚拟上下文、策略学习管理
- 涵盖写路径过滤、矛盾处理、延迟预算、隐私治理等工程实现挑战
- 覆盖个人助手、编程 Agent、开放世界游戏、科学推理、多 Agent 协作五大应用场景

📊 **实验结果**
分析了四个近期 Agentic 记忆基准，揭示当前系统在跨会话连续整合和因果检索方面的显著短板。

🎯 **与 Agent/Routing 的关联**
为 Agent 记忆系统设计提供了权威分类参考框架，对构建具有可扩展记忆的多 Agent 系统有直接指导价值。

## 📄 Paper 3

📄 **[高效 LLM 推理的动态模型路由与级联综述]**（Dynamic Model Routing and Cascading for Efficient LLM Inference: A Survey）
🔗 https://arxiv.org/abs/2603.04445

💡 **核心贡献**
首篇系统性分析跨多个独立训练 LLM 的路由与级联方法的综述，区别于 MoE 内部路由，专注于不同模型间的查询分派决策；提出三维概念框架（决策时机 × 使用信息 × 计算方式），揭示实际系统的组合性本质。

🔧 **技术亮点**
- 覆盖查询难度路由、人类偏好路由、聚类路由、不确定性量化、强化学习路由、多模态路由、级联路由七大范式
- 分析每种范式的代表方法及关键 trade-off（性能 vs 成本 vs 延迟）
- 指出精心设计的路由系统可通过跨模型专长复用，超越单一最强模型的表现

📊 **实验结果**
通过跨范式对比分析，量化说明不同路由策略在不同部署约束下的适用性差异；展示组合路由系统在效率增益上的潜力。

🎯 **与 Agent/Routing 的关联**
直接系统化了 LLM Routing 领域知识图谱，是当前 Routing 研究最权威的参考综述之一。

## 📄 Paper 4

📄 **[MoE 路由测试床：小规模专家特化与路由行为研究]**（MoE Routing Testbed: Studying Expert Specialization and Routing Behavior at Small Scale）
🔗 https://arxiv.org/abs/2604.07030

💡 **核心贡献**
提出 MoE Routing Testbed，通过配对「可区分域数据集 + 理想路由参考器」的方式，在小规模实验中获得对路由动态的清晰量化视角；发现「平衡范围（balancing scope）」是允许专家特化同时保持高利用率的关键因素。

🔧 **技术亮点**
- 构建明确上界参考路由器，为专家特化程度提供可量化评估基准
- 系统比较多种 MoE 路由方法，验证 balancing scope 在小规模下的决定性作用
- 实验证明小规模结论可泛化至 35× 更大模型规模

📊 **实验结果**
通过可控测试床，量化了各路由方法的专家特化程度，并以参考路由器为上界进行对比；发现 balancing scope 设计可显著提升专家利用率与特化质量，且规律在大模型上一致成立。

🎯 **与 Agent/Routing 的关联**
深化了对 MoE 内部路由机制的理解，对设计高效 Agent 推理系统中的专家选择策略具有直接参考价值。

## 🧠 今日洞察

**Agent 记忆走向端到端可优化**：从辅助模块到强化学习直接优化的记忆管理策略（AgeMem）标志着 Agent 记忆已进入「策略即记忆」新阶段，传统 RAG+规则模式正在被学习型记忆管理取代。

**Routing 研究双线并进**：跨模型路由（多 LLM 路由综述）与模型内路由（MoE Testbed）正在同步深化——前者关注如何在异构模型间智能分配查询，后者关注如何让模型内部专家真正实现差异化特化，两者都指向「更细粒度的计算资源按需分配」这一核心命题。

**评估范式正在重构**：无论是 Agent 记忆领域从静态 Recall 到多会话 Agentic 基准的迁移，还是 MoE Routing 从感知评估到可量化测试床的升级，都反映出研究社区正在从「能力展示」转向「系统性可测量」，这将加速实际可部署系统的迭代速度。
