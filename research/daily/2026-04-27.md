### 1️⃣

📄 **【综述】LLM 自主 Agent 的记忆机制：机制、评估与前沿**（Memory for Autonomous LLM Agents: Mechanisms, Evaluation, and Emerging Frontiers）
🔗 https://arxiv.org/abs/2603.07670

💡 **核心贡献**
将 Agent 记忆形式化为 **写入-管理-读取（write-manage-read）** 循环，系统梳理了从 2022 年到 2026 年初 LLM Agent 记忆的设计、实现与评估全貌；提出了覆盖时间范围、表示底层与控制策略三个维度的分类体系，并分析了评估范式从静态召回到多会话 Agent 测试的演变。

🔧 **技术亮点**
- 归纳五大记忆机制族：上下文压缩、检索增强存储、反思自改进、层级虚拟上下文、策略学习管理
- 分析了写路径过滤、矛盾处理、延迟预算和隐私治理等工程实现挑战
- 指出持续整合、因果检索、可信反思、习得遗忘与多模态具身记忆是主要开放挑战

📊 **实验结果**
覆盖四个近期 Benchmark，揭示当前系统在多会话决策与记忆交织场景中存在持续性性能差距。

🎯 **与 Agent/Routing 的关联**
系统梳理了 Agent 长期记忆的全栈技术栈，是理解 Agent 状态管理与跨会话规划的核心参考综述。

### 2️⃣

📄 **HiMem：LLM 长时序 Agent 的分层长期记忆**（HiMem: Hierarchical Long-Term Memory for LLM Long-Horizon Agents）
🔗 https://arxiv.org/abs/2601.06377

💡 **核心贡献**
提出 HiMem 框架，通过"话题感知事件-惊奇度双通道分割"构建情节记忆（Episode Memory），并通过多阶段信息抽取流水线构建笔记记忆（Note Memory），两者语义连接形成分层结构，支持高效检索与持续自进化。

🔧 **技术亮点**
- 双通道分割策略（主题感知 × 惊奇度信号）精准切割对话片段为情节单元
- 混合检索与尽力检索（hybrid / best-effort retrieval）策略平衡精度与效率
- 冲突感知记忆再巩固（Conflict-aware Memory Reconsolidation）实现知识动态修正

📊 **实验结果**
在长时序对话 Benchmark 上一致优于代表性基线，在准确性、一致性和长期推理三项指标均有显著提升，且效率保持优良（代码已开源：https://github.com/jojopdq/HiMem）。

🎯 **与 Agent/Routing 的关联**
直接解决 Long-Horizon Agent 的记忆瓶颈，为需要跨多步规划的 Agent 提供可扩展的分层记忆范式。

### 3️⃣

📄 **长时序企业 AI Agent 的四轴决策对齐**（Four-Axis Decision Alignment for Long-Horizon Enterprise AI Agents）
🔗 https://arxiv.org/abs/2604.19457

💡 **核心贡献**
提出将长时序决策行为分解为四个正交对齐轴（FRP、RCS、CRR、CAR）的评估框架，揭示聚合准确率指标会掩盖各维度失效模式；在覆盖贷款审批和保险理赔的受控 Benchmark（LongHorizon-Bench）上对六种记忆架构进行对比，发现聚合指标与轴级指标结论存在显著逆转。

🔧 **技术亮点**
- **FRP（事实精度）/ RCS（推理一致性）/ CRR（合规重建）/ CAR（校准弃权）** 四轴独立可测
- CRR 是新颖的监管约束轴，CAR 将覆盖率与准确率解耦——均未被现有对齐文献充分研究
- 框架迁移性强：仅需构建事实 Schema + 校准 CRR 审计提示即可适配任意受监管决策域

📊 **实验结果**
六种记忆架构对比：检索型架构在 FRP 上坍塌；Schema 锚定架构付出结构化税；朴素摘要+事实保全 Prompt 在 FRP/RCS/CRR 上表现最强；所有架构在每个案例上均不弃权，暴露了 CAR 轴的系统性失效。（代码：https://github.com/vasundras/decision-alignment-long-horizon-agents）

🎯 **与 Agent/Routing 的关联**
对需要 Routing 到不同记忆/执行架构的企业 Agent 至关重要，四轴分解可直接指导路由策略的选择与评估。

### 4️⃣

📄 **高效 LLM 推理的动态模型路由与级联：综述**（Dynamic Model Routing and Cascading for Efficient LLM Inference: A Survey）
🔗 https://arxiv.org/abs/2603.04445

💡 **核心贡献**
系统分析了跨多个独立训练 LLM 的动态路由与级联方法（区别于 MoE 内部路由），覆盖查询难度、人类偏好、聚类、不确定性量化、强化学习、多模态、级联等七大范式；提出了从"决策时机-使用信息-计算方式"三维刻画路由系统的概念框架。

🔧 **技术亮点**
- 跨模型路由与 MoE 内路由的明确区分，强调实践系统往往是多范式组合
- 不确定性量化与强化学习范式在路由中的应用深度分析
- 指出实际部署下多目标权衡：成本、延迟、能力、泛化性四者难以同时最优

📊 **实验结果**
综述分析表明：设计合理的路由系统可超越单一最强模型，通过跨模型能力组合实现效率与质量的双赢；但路由机制在跨架构、跨模态泛化上仍是开放挑战。

🎯 **与 Agent/Routing 的关联**
直接覆盖 LLM Routing 核心主题，为构建多 Agent 协同推理系统中的模型选择层提供了全面的方法论地图。

## 🧠 今日洞察

1. **记忆从"组件"升级为"基础设施"**：从 HiMem 的分层架构到 Memory 综述的全栈梳理，2026 年的 Agent 记忆研究已从单一模块走向系统工程——写路径过滤、冲突消解、延迟预算缺一不可，记忆正成为 Agent 能力的核心基础设施而非附加功能。

2. **评估粒度决定改进方向**：LongHorizon-Bench 的四轴分解揭示，聚合准确率会掩盖关键失效（如合规缺失、决策不弃权），这意味着 Agent 路由与记忆架构的选型需要细粒度的多轴评估，而非单一胜率指标。

3. **Routing 研究加速成熟**：动态模型路由已从简单的"难题走大模型"演进到融合 RL、不确定性量化、多模态的复合范式，并被证明可超越单一最强模型——这预示 Agent 系统中的"智能调度层"将成为下一个竞争焦点。
